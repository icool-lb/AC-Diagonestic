## What Wave 5 adds
Wave 5 turns the code list into a **technician-oriented diagnostic tool**:
- Search by **code**
- Search by **symptom**
- Rule-based **smart suggestion engine**
- **Illustrated diagnostic categories**
- Bilingual **Arabic + English** presentation
- Strong source traceability to the underlying OEM/public manuals

## Important evidence note
The **code meanings** remain tied to the original source records.  
The new fields `likely_causes`, `check_steps`, `parts_suspects`, and AI hints are **field guidance** inferred from:
1. the official code meaning in the linked source, and
2. standard HVAC troubleshooting practice.

They are meant to help technicians move faster, but they **do not replace the exact service manual for the model in hand**.
